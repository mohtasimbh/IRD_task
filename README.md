# IRD Mobile App Testing 

This is UI model of the Task App. 

## Deployment

To deploy this project run

```bash
  flutter pub get
  open -a simulator (to get iOS or andriod Simulator)
  flutter run
  flutter run -d chrome --web-renderer html (to see the best output)
```
